WELCOME TO THE WINDOWS 8 DESIGN HANDBOOK

The recent launch of Windows 8 brings a lot of new opportunities for both designers and developers to build beautiful apps that are fun and easy to use. Windows 8 introduced a new design language, and there is still some confusion about the best ways to follow that design language. This site is a collection of explanations of the most important aspects of the new Windows 8 design language.
